# TikoBot_tg
Use different LLM on Telegram.

## 可用的文字处理模型
Gemini (Gemini-pro)\
GPT-3.5 ([ChatAnywhere](https://github.com/chatanywhere/GPT_API_free)) \
Qwen (Qwen-plus)

## 可用的图片处理模型
Gemini (Gemini-pro-vision)

## 使用方法
1. 配置 config.ini。

2. 运行
```
cd ./TikoBot_tg
pip install  -r requirements.txt
python  TikoBot_tg.py
```
